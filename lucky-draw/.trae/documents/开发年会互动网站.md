# 年会互动网站开发计划

## 技术栈选择
- **前端**：React + Vite + Tailwind CSS
- **后端**：Node.js + Express
- **数据库**：MongoDB
- **实时通信**：Socket.io
- **认证**：JWT

## 项目结构

### 前端 (`frontend/`)
- `src/components/` - 可复用组件
  - `MemberList/` - 成员列表组件
  - `LuckyDraw/` - 抽奖组件
  - `RedEnvelope/` - 红包动画组件
  - `Animation/` - 春节动画效果组件
- `src/pages/` - 页面
  - `Home/` - 首页
  - `Login/` - 登录/注册页
  - `Admin/` - 后台管理页
- `src/services/` - API 服务
- `src/hooks/` - 自定义 hooks
- `src/utils/` - 工具函数
- `src/assets/` - 静态资源

### 后端 (`backend/`)
- `src/controllers/` - 控制器
  - `userController.js` - 用户管理
  - `lotteryController.js` - 抽奖逻辑
  - `adminController.js` - 后台管理
- `src/models/` - 数据模型
  - `User.js` - 用户模型
  - `Lottery.js` - 抽奖记录模型
  - `RedEnvelope.js` - 红包配置模型
- `src/routes/` - 路由
- `src/middleware/` - 中间件
  - `auth.js` - 认证中间件
  - `errorHandler.js` - 错误处理
- `src/socket/` - WebSocket 处理
- `src/config/` - 配置
- `src/utils/` - 工具函数

## 核心功能实现

### 1. 成员扫码加入系统
- 生成包含注册链接的二维码
- 实现用户快速注册与登录
- 使用 Socket.io 实时更新成员状态
- 展示成员列表及在线状态

### 2. 随机抽奖红包功能
- 开发点击触发式随机抽奖系统
- 实现红包动态掉落动画效果
- 支持自定义红包金额范围、中奖概率及奖品设置
- 记录并展示中奖历史及排行榜

### 3. 前端界面设计
- 采用喜庆的红色为主色调，融入春节元素
- 设计响应式界面，适配不同设备
- 实现春节相关动画效果（烟花、灯笼、爆竹等）
- 添加流畅的交互动画及过渡效果

### 4. 后端 API 设计
- **用户管理**：注册、登录、获取成员列表
- **抽奖管理**：触发抽奖、获取中奖记录、获取排行榜
- **红包配置**：创建/更新红包配置、获取配置
- **后台管理**：活动参数配置、数据统计

### 5. 数据库设计
- **User**：姓名、部门、手机号、状态、创建时间
- **Lottery**：用户 ID、奖品类型、金额、时间、状态
- **RedEnvelope**：金额范围、中奖概率、奖品设置、创建时间

### 6. 实时通信
- 使用 Socket.io 实现成员状态实时更新
- 实现抽奖结果实时广播
- 支持红包掉落动画的实时同步

### 7. 安全与性能优化
- 实现 JWT 身份验证
- 确保抽奖过程的公平性与随机性
- 优化前端加载速度及动画性能
- 设计高并发场景下的系统稳定性方案

### 8. 部署与维护
- 提供完整的部署文档及环境配置说明
- 实现后台管理界面，支持活动参数配置与数据统计
- 设计系统监控与异常处理机制

## 开发步骤
1. 初始化项目结构
2. 搭建后端基础架构
3. 设计并实现数据库模型
4. 开发核心 API 接口
5. 实现 WebSocket 实时通信
6. 搭建前端基础架构
7. 开发核心前端组件
8. 实现春节主题 UI 设计
9. 开发动画效果
10. 集成前后端
11. 测试与优化
12. 部署与文档编写
